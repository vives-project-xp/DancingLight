alsa is de sounddriver in linux

hier is welke files je moet veranderen om onze code te kunnen runnen:

in /etc/asound.conf en /home/pi/.asoundrc moet dit allebij komen:

pcm.!default {
  type asym
  capture.pcm "mic"
}
pcm.mic {
  type plug
  slave {
    pcm "hw:1,0"
  }
}

Volgende lijn moet komen in /etc/modprobe.d/alsa-base.conf:

options snd slots=snd-bcm2835,snd-usb-audio,vc4,vc4

En dan reboot
Je moet er ook nog voor zorgen dat je pyaudio hebt geinstalleerd, er zijn nog wat extra paketten die we hebben
geinstalleerd op onze pi maar we weten niet ofdat deze werkelijk nodig zijn voor de uitvoer van onze code 
(we voegden ze toe omdat dit in een repo van onze leeraar zat (https://github.com/ronnymees/audiorecorder_with_pi 
en dat we hoopten dat het hiermee wel ging) als het niet werkt kan je proberen deze nog te installeren:

sudo apt-get install libportaudio0 libportaudio2 libportaudiocpp0 portaudio19-dev
